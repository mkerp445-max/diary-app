# 体調管理日記アプリ

## 1. 概要
日々の体調、気分、睡眠時間、日記を記録し、自分のコンディションを分析するためのツールです。

## 2. 開発した理由
自身の生活リズムと気分の関係を客観的に知り、健康管理に役立てたいと考えたため

## 3. このアプリでできること
- 8桁の日付（例：20260404）での日記の保存・検索・削除
- 就寝・起床時刻からの自動睡眠時間計算
- 蓄積したデータに基づいた「体調と気分の相関関係」の分析レポート出力
- 直近5日間のコンディションを「■□」のグラフで視覚的に表示

## 4. こだわったポイント（工夫した点）
- **データの整合性**: ユーザーが間違えて文字を入力したり、存在しない日付を入れないようエラーチェック機能を充実させました。
- **効率的な計算**: Pandasライブラリの `apply` や `lambda` を使い、大量のデータを一括で処理できるようにしました。
- **見やすさ**: 統計画面では、小数点第2位までの表示やグラフ表示など、パッと見て状況がわかるように工夫しました。

## 5. 使用した技術
- 言語: Python
- ライブラリ: Pandas (データ操作), Datetime (時間計算), OS (ファイル管理)
