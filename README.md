# 体調管理日記アプリ

## 1. 概要
日々の体調・気分・睡眠時間・日記を記録し、コンディションを分析するCLIアプリです。

## 2. 開発した理由
生活リズムと気分の関係を可視化し、自己管理に活かすために作成しました。

## 3. 主な機能
・日記の保存／検索／削除（8桁の日付で管理）  
・就寝・起床時刻からの睡眠時間自動計算  
・体調・気分・睡眠時間の統計分析（平均・相関）  
・直近5日間のコンディションの可視化表示  

## 4. 工夫した点
・入力バリデーションを実装し、誤入力（不正な日付・時間・数値）を防止  
・処理を関数単位に分割し、可読性と再利用性を意識  
・Pandasを用いてデータ集計・分析処理を効率化  

## 5. 使用技術
・Python  
・Pandas  
・Datetime  
・OS
